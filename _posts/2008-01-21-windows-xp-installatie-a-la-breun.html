Ik heb pas nog een Mac compleet opnieuw geïnstalleerd, maar afgelopen weekend was het alweer tijd voor een herinstallatie van een Windows XP-machine. De harde schijf was namelijk overleden. Dit keer maakte ik geen aantekeningen, maar volgens mij heb ik het grootste deel wel onthouden. Het doel was een computer voor mijn tante en haar twee schoolgaande kinderen. Een Windows-licentie was aanwezig, maar verder graag alle software gratis (en legaal).

  1. Vervang de harde schijf (en prik en passent wat extra RAM bij)
  2. Installeer Windows XP
  3. Schakel de Windows Firewall in
  4. Installeer de AVG Free virusscanner
  5. Installeer het laatste Windows service pack
  6. Via Windows Update schakel je Microsoft Update in
  7. Microsoft Update tot je erbij neervalt
  8. Installeer benodigde drivers voor videokaart, printer en geluidskaart (eventueel SiSoft Sandra Lite installeren om uit te zoeken watvoor hardware er in de computer zit)
  9. Schakel het groeperen van iconen in de taakbalk uit
  10. Schakel systeemherstel uit
  11. Zorg ervoor dat extensies voor bekende bestandsformaten niet verborgen worden
  12. Gratis software installeren
    • Mozilla Firefox met de IE Tab add-on
    • Adobe Flash Player
    • Sun Java Runtime Environment
    • Mozilla Thunderbird ((Gmail account aanmaken en) accounts configureren)
    • OpenOffice.org (instellen op standaard opslaan als .doc en .xls voor documenten en spreadsheets voor compatibiliteit met school)
    • Picasa
    • Paint.NET
    • K-Lite Codec Pack Standard (voor Xvid, DivX e.d.)
    • Skype
    • Windows Live Messenger
    • Hitman Pro
    • Google Earth
    • 7-Zip

Dit alles kostte mij een hele middag.
